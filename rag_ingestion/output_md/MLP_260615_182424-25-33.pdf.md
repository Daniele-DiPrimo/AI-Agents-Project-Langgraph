# Backpropagation

## Introduzione alla Backpropagation

La backpropagation è l'algoritmo utilizzato per calcolare efficientemente i gradienti necessari per la discesa del gradiente.

### Fasi della Backpropagation

*   **Forward Pass:** Calcolo dell'output passando l'input attraverso la rete.
*   **Calcolo della Perdita:** Confronto dell'output con il valore atteso tramite una funzione di costo (ad esempio, entropia incrociata o errore quadratico medio).
*   **Backward Pass:** Applicazione della regola della catena per calcolare il gradiente della perdita rispetto a ciascun peso e bias.
*   **Aggiornamento dei Pesi:** I pesi sono aggiornati utilizzando il gradiente calcolato e la SGD (Stochastic Gradient Descent).

## Forward Pass

$$Z1 = np.dot(X\_train, W1) + b1$$
$$A1 = sigmoid(Z1)$$
$$Z2 = np.dot(A1, W2) + b2$$
$$A2 = sigmoid(Z2)$$

*   $X\_train$: matrice degli input di training, dimensione ($n_{samples}, n_{input\_features}$)
*   $W_1$: pesi del primo layer (input $\rightarrow$ hidden), dimensione ($n_{input\_features}, n_{hidden}$)
*   $b_1$: bias del primo layer, dimensione (1, $n_{hidden}$)
*   $Z_1$: input lineare al primo layer
*   $A_1$: attivazioni del primo layer (dopo applicazione della funzione sigmoide)
*   $W_2$: pesi del secondo layer (hidden $\rightarrow$ output), dimensione ($n_{hidden}, n_{output}$)
*   $b_2$: bias del secondo layer
*   $Z_2$: input lineare al secondo layer
*   $A_2$: output finale della rete, usato come predizione
*   $\sigma(x)$: funzione sigmoide, definita come $\sigma(x) = \frac{1}{1+e^{-x}}$

## Funzione di Loss: Cross-Entropy

$$loss = -\frac{1}{N} \sum_{i=1}^{N} [y_{train}^{(i)} \cdot \log(A_2^{(i)} + \epsilon) + (1 - y_{train}^{(i)}) \cdot \log(1 - A_2^{(i)} + \epsilon)]$$

*   $N$ è il numero di campioni nel dataset di training.
*   $y_{train}^{(i)}$ è l'etichetta vera (target) del campione $i$, che può essere 0 o 1 (classificazione binaria).
*   $A_2^{(i)}$ è la predizione della rete neurale per il campione $i$, che è il valore di output del modello, compreso tra 0 e 1 grazie alla funzione sigmoide.
*   $\epsilon$ è un piccolo valore (ad esempio $1e^{-8}$) aggiunto per evitare il logaritmo di zero che è indefinito.

$$loss = -\text{np.mean}(y\_{train} * \text{np.log}(A_2 + 1e-8) + (1 - y\_{train}) * \text{np.log}(1 - A_2 + 1e-8))$$

## Backward Pass: Calcolo dei Gradienti

$$dZ2 = A2 - y\_train$$
\# Calcolo dell'errore nel layer di output:

$$dW2 = \text{np.dot}(A1.T, dZ2) / X\_train.shape[0] \quad \# \text{Calcolo del gradiente dei pesi } (W\_2) \text{ nel secondo layer}$$
$$db2 = \text{np.sum}(dZ2, axis=0, keepdims=True) / X\_train.shape[0] \quad \# \text{Calcolo del gradiente dei bias } (b\_2) \text{ nel secondo layer}$$

$$dZ1 = dA1 * \text{sigmoid\_derivative}(Z1) \quad \# \text{Calcolo dell'errore nel primo layer (hidden layer)}$$
$$dW1 = \text{np.dot}(X\_train.T, dZ1) / X\_train.shape[0] \quad \# \text{Calcolo del gradiente dei pesi } (W\_1) \text{ nel primo layer}$$
$$db1 = \text{np.sum}(dZ1, axis=0, keepdims=True) / X\_train.shape[0] \quad \# \text{Calcolo del gradiente dei bias } (b\_1) \text{ nel primo layer}$$

Similmente a $db2$, il gradiente per i bias è la somma dell'errore $dZ1$ nel primo layer, media con il numero di campioni.

*   Questo valore verrà utilizzato per aggiornare i bias del primo layer.
*   $W1 -= \text{learning\_rate} * dW1$
*   $W2 -= \text{learning\_rate} * dW2$
*   $b2 -= \text{learning\_rate} * db2$

## Modifica per la Classificazione Multiclass

### Cosa fare per il multiclass?

*   Per modificare la rete neurale da una versione binary classification a una multiclass classification, sono state apportate alcune modifiche chiave nel codice.
*   Queste modifiche riguardano principalmente la funzione di attivazione nell'output layer, la funzione di perdita (loss function), e la gestione dei gradienti.

### Funzione di attivazione per il layer di output: Softmax

Data una lista di valori reali (detti logits):
$$z = (z_1, z_2, ..., z_K)$$

la funzione softmax restituisce una lista di valori:
$$\text{softmax}(z_i) = \frac{e^{z_i}}{\sum_{j=1}^{K} e^{z_j}}$$

### Funzione di perdita: Categorical Cross-Entropy

$$L = -\sum_{i=1}^{K} y_i \log(p_i)$$

*   Motivo: La derivata combinata tra softmax e categorical cross-entropy è diretta e semplifica il calcolo durante il backward pass.
